Introduction

There are tons of options for utilizing various fonts on your website. One of the simpletst methods to utilize non-default fonts is to use Google Fonts, a font library populated with nearly one thousand fonts to choose from. This guide will walk you through selecting, embedding, and styling Google Fonts for your website.

Prior Knowledge

This guide assumes you already have the following skills:

  • Basic computer usage skills
  • A general understanding of the Internet
  • A solid working knowledge of HyperText Markup Language (HTML)
  • A basic foundational knowledge of Cascading Style Sheets (CSS)
Searching Google Fonts

Google Fonts offers an intuitive search that can be filtered in helpful ways. If you know the name of the font you are searching for, simply enter it in the search field.

If you do not know the name of the font you want, or if you prefer to browse, Google Fonts offers filters to help you pinpoint the exact style you are looking for. Use the Categories filter to choose between serif and sans serif fonts, among other options. Use the Language filter to choose fonts suited to languages other than the default English. Use the Font Properties filters to select for other characteristics such as thickness, slant, or width.

Linking to Fonts

After searching Google Fonts and finding the font you would like to use, select it by clicking the plus sign in the top right corner of the font's box. The font family will be added to a pop up window in the lower right corner. Inside this pop up window you can view the code used to add the font to your site. Before copying and pasting the code, select the customize tab. From here you can select the options you would like embedded with your font. You can select from varoius weights (such as bold and light) and styles (such as italic).

Once you have selected all the options you will need, click on the embed tab. You will see two snippets of code. The first code snippet should be copied and pasted into the head section of your website. This will allow your site to pull the font in for utilization in the browser when people visite your website. The code below shows the font used for the headlines on this page:

<link href="https://fonts.googleapis.com/css?family=Lato:300,400,400i,700&display=swap" rel="stylesheet">
Selecting Fonts with CSS

After linking to your chosen fonts, it is time to put them to use. Use CSS to select the elements you wish to apply your fonts to. Once again, this code can be found in the pop up window in Google Fonts. The CSS code shown below will apply the Lato font to an h1 element:

h1 {
font-family: 'Lato', sans-serif;
}
Font Size & Font Weight

Font Size

Selecting font sizes with CSS uses the font-size property. The actual font size can be denoted in three different ways.

  • Absolute sizing uses a defined px number to set font-size.
    h1 {
    font-size:60px
    }
  • The em unit sets font size of a child element proportional to the font size of the nearest parent element.
    h1 {
    font-size:3em
    }
  • The rem unit sets font size of an element proportional to the base font size set at the root of the page (The base font size is set in the HTML selector in CSS using absolute sizing).
    h1 {
    font-size:3rem
    }

Font Weight

Font weight is the thickness of your font. It can range from very light to very bold. Font weight can be set using the CSS property font-weight. Font weight can be set using two types of values. The following examples will use the HTML:

<h1>Hello world!</h1>
  • Font weight can be set numerically (100 - 900). Higher numbers denote heavier font-weights.
    h1 {
    font-weight:300
    }

    Hello world!

  • Font weight can be set semantically using terms like light, regular, and bold.
    h1 {
    font-weight:bold
    }

    Hello world!

Font Style & Font Variant

Font Style

The font-style property allows you to display selected text in various styles using the values normal, italic, and oblique. The following examples will use the HTML:

<h1>Hello world!</h1> h1 {
font-style:italic
}

Hello world!

Font Variant

The font-variant property allows you display selected text in a variation other than the default. This typically indicates usage of the small-caps value. The CSS below will apply styles to the following HTML:

<h1>Hello world!</h1> h1 {
font-variant:small-caps
}

Hello world!

Text Transform

The text-transform property transforms selected text based on several values. The following examples will use the HTML:

<h1>Hello world!</h1>
  • none : This is the default value. Text will render exactly as it is entered in HTML.
    h1 {
    text-transform:none;
    }

    Hello world!

  • capitalize : This will transform the text such that the first letter of each word is capitalized.
    h1 {
    text-transform:capitalize;
    }

    Hello world!

  • uppercase : This value will transform all characters into uppercase.
    h1 {
    text-transform:uppercase;
    }

    Hello world!

  • lowercase : This value will transform all characters into lowercase.
    h1 {
    text-transform:lowercase;
    }

    Hello world!

Color

Changing the color of your text via CSS is simple. Just use the color property. The value you can use to change the color, however, can be written in three different ways. The following examples will use the HTML:

<h1>Hello world!</h1>
  • Web Safe Colors : These semantically named colors use commonly known color names as values, such as red, teal, or seagreen.
    h1 {
    color:crimson;
    }

    Hello world!

  • HEX Values  : This value uses a pound sign (#) followed by a six-digit code that represents a specific color.
    h1 {
    color:#435fef;
    }

    Hello world!

  • RGB Values: This value uses a series of three numbers, each representing the amount of red, green, and blue present in the color.
    h1 {
    color:rgb(235, 95, 25);
    }

    Hello world!
